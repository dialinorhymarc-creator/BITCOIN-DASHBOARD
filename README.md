# 🪙 Crypto Bitcoin Dashboard

A real-time data dashboard built with Python and Streamlit to monitor and analyze Bitcoin and cryptocurrency metrics. This application pulls data, processes it using Pandas, and presents it through an interactive, user-friendly web interface.

---

## 🚀 Features

* **Interactive Data Visualization:** Dynamic charts and graphs tracking Bitcoin trends.
* **Database Integration:** Efficient data retrieval powered by Pandas and SQLAlchemy.
* **Clean UI/UX:** Built on top of Streamlit for a fast, responsive, and modern dashboard experience.

---

## 🛠️ Tech Stack

* **Frontend/Framework:** [Streamlit](https://streamlit.io/)
* **Data Manipulation:** [Pandas](https://pandas.pydata.org/)
* **Database Engine:** [SQLAlchemy](https://www.sqlalchemy.org/)
* **Language:** Python 3.14+
